## Running on Locally

Download the code.

```bash
$ git clone https://github.com/First8Memristor/f8.git
```

Set up db(mongodb)

mongodb://66.94.102.219:27017/f8_database

```bash
$ cd f8-backend
$ yarn backend
```
```bash
$ cd f8-frontend
$ yarn dev
```
